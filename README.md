# Sistema de Gestión de Alumnos

Proyecto desarrollado en JavaScript como evaluación del módulo.

## Descripción

La aplicación permite administrar un curso de alumnos mediante una interfaz web sencilla.
Cada alumno posee:

- ID
- Nombre
- Apellido
- Promedio

## Funcionalidades

- Agregar alumnos.
- Buscar alumnos por ID.
- Eliminar alumnos.
- Mostrar estadísticas del curso.
